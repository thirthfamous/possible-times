# possible-times

Simple app that count of possibility of times from an array of integer

---

### Running the App

```sh
// clone the project
git clone https://github.com/thirthfamous/possible-times

// go to the project directory
cd possible-times

// install the dependencies
go get

// run the tests
go test \test
or
go test .\test\

// run the main
go run main.go
```
